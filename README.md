# WireQuokka
---

## Features

- Realtime packet sniffing
- Plaintext credential detection
- ARP spoof detection
- DNS monitoring
- Port scan detection
- Live terminal dashboard
- Packet logging
- Modular architecture

---

## Technologies

- Python
- Scapy
- Rich
- OSI Layer 2/3 packet analysis

---

## Installation

```bash
pip install -r requirements.txt
```

---

## Run

Linux/macOS:

```bash
sudo python main.py
```

Windows:

```powershell
python main.py
```

---

## Logs

Alerts are automatically saved into:

```text
logs/alerts.log
```
